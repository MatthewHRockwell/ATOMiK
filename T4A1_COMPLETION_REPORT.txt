SCHEMA FEATURES IMPLEMENTED
================================================================================

Section A: Catalogue (Positioning Metadata)
----------------------------------------------------------------------
- vertical: 8 predefined options (Video, Edge, Network, Finance, etc.)
- field: Domain-specific subcategory (PascalCase, 1-64 chars)
- object: Component name (PascalCase, must be unique)
- version: Semantic versioning (semver format)
- author: Optional author/organization
- license: Optional SPDX license identifier
- description: Human-readable summary

Section B: Schema (Computational Definition)
----------------------------------------------------------------------
Delta Fields:
- 3 types: delta_stream, bitmask_delta, parameter_delta
- Widths: 8, 16, 32, 64, 128, 256 bits (powers of 2)
- Encodings: spatiotemporal_4x4x4, raw, rle
- Compression: xor, rle, none
- Optional default values

Operations:
- accumulate (required): XOR-based delta composition
- reconstruct (optional): State reconstruction
- rollback (optional): Temporal state reversal with history

Constraints:
- max_memory_mb: Memory budget (1-65536 MB)
- max_power_mw: Power budget (1-100000 mW)
- update_latency_ms: Latency requirement (0-10000 ms)
- target_frequency_mhz: Hardware frequency (1.0-1000.0 MHz, default: 94.5)

Section C: Hardware Mapping (Optional)
----------------------------------------------------------------------
- target_device: FPGA family/part number
- rtl_params: DATA_WIDTH, ENABLE_PARALLEL, CLOCK_DOMAIN
- synthesis_options: optimization_goal (area/speed/power)
