Architecture

delta_in ──> Round-Robin Distributor ──> Bank[0..N-1] ──> XOR Merge Tree ──> current_state
                                         (N x atomik_delta_acc)   (log2(N) depth)

  Each bank:
    LOAD:       initial_state <- data_in
    ACCUMULATE: accumulator <- accumulator XOR data_in     (single cycle, no carry)
    READ:       data_out <- initial_state XOR accumulator   (combinational)
